## Crypto модуль (учёт крипто‑баланса)

Этот модуль реализует безопасный учёт крипто‑баланса пользователя для Laravel:
- **раздельные поля**: `available` (доступно) и `locked` (заморожено)
- **журнал проводок (ledger)** с **идемпотентностью** (`idempotency_key`)
- изменения баланса только через **транзакции БД** и `SELECT ... FOR UPDATE`
- сценарии блокчейна: **асинхронность, подтверждения, повторы, реорги**

### Что внутри

- `src/Models/*`:
  - `Wallet` — агрегат баланса пользователя по валюте
  - `WalletLedger` — проводки (дельты по `available/locked`)
  - `Withdrawal` — заявки на вывод
- `src/Services/*`:
  - `WalletBalanceService` — атомарные операции `credit / debit / lock / unlock`
  - `DepositService` — обработка депозитов после N подтверждений
  - `WithdrawalService` — создание/подтверждение/отмена вывода с заморозкой средств
- `database/migrations/*` — миграции таблиц
- `config/crypto.php` — базовые настройки (подтверждения, лимиты)
